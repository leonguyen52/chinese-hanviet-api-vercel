Chinese-Han Viet Cognates API

A modern API for looking up Chinese and Han Viet cognates

Overview

An API built using json-server that allows you to look up Chinese and Han Viet cognates. This project, originally started by ryanphung, has been updated with a new database to match specific requirements.
